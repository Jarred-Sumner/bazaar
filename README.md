# Bazaar

Bazaar is a random item generator.

## Installation

Add this line to your application's Gemfile:

    gem 'bazaar'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bazaar

## Usage

    Bazaar.item
    => "Bike"
